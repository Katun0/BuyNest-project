🛒 BuyNest – Plataforma de Compras Online com Symfony

BuyNest é uma aplicação web desenvolvida com Symfony focada em
vendas online no estilo marketplace. A plataforma permite que usuários
naveguem por produtos, adicionem itens ao carrinho e realizem pedidos,
tudo com autenticação de conta integrada.

________________________________

🚀 Funcionalidades Previstas

Cadastro e login de usuários autenticados

Listagem de produtos organizados por categorias

Sistema de carrinho de compras

Finalização de pedidos (checkout)

Painel de administração para gerenciamento de produtos e usuários (em
desenvolvimento)

Integração futura com gateways de pagamento

________________________________

🧰 Tecnologias Utilizadas

Symfony 7.11

PHP 8.x

Doctrine ORM

Twig (motor de templates)

Bootstrap 5 (para o front-end)

Firebase

Composer (gerenciador de dependências PHP)

________________________________

⚙️ Instalação e Execução

Pré-requisitos

PHP 8.1 ou superior

Composer

Symfony CLI (opcional, mas recomendado)
